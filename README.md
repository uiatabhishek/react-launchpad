# 🚀 React Launchpad v3

## ⚡ Production Ready React Boilerplate

A modern React starter with Vite, Tailwind, Zustand, Axios, and React Query.

---

## 🔥 Features
- Zustand state management
- Axios API layer
- React Query for server state
- Clean architecture
- Tailwind styling

---

## 🚀 Setup
```bash
npm install
npm run dev
```

---

## 🌍 Env
Create .env file:
VITE_API_BASE_URL=https://api.example.com

---

## 📁 Structure
- store/
- services/
- providers/
- pages/
